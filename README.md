# RAG Document Intelligence System

A Retrieval-Augmented Generation web app for uploading documents (PDF, DOCX, TXT) and querying them in natural language with cited, grounded answers.

Built with FastAPI, LangChain, ChromaDB, sentence-transformers, Streamlit, and the Anthropic API. See [CLAUDE.md](CLAUDE.md) for developer commands and conventions. Full README (architecture diagram, deployment, demo) lands in Week 3.
